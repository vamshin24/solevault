# SoleVault | Automation shouldn’t stop at shoes.

SoleVault is a smart cleaning robot designed to manage footwear obstacles in real-time, ensuring that autonomous floor cleaning remains uninterrupted. Built for modern homes with kids, pets, and daily clutter.

## 🚀 Key Features

- **Footwear Detection**: Advanced sensors identify shoes in the cleaning path.
- **Obstacle Management**: Safely clears footwear to complete the cleaning cycle.
- **Smart-Home Integration**: Designed to work seamlessly in smart-home environments.
- **Responsive Design**: A premium, one-page landing site built with HTML, CSS, and vanilla JS.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern layout with Flexbox, Grid, and Glassmorphism.
- **JavaScript**: Smooth transitions and form handling.
- **Lucide Icons**: Clean, scalable vector icons.
- **Google Fonts (Inter)**: Premium typography.

## 📦 Local Development

To view the site locally:
1. Clone this repository.
2. Open `index.html` in any modern web browser.
